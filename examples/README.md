# Examples

- **mailagent.minimal.yml** — Minimal single-inbox config to get started
- **mailagent.yml** — Full-featured multi-inbox config with all options demonstrated
- **mailagent.test.yml** — Test definitions for `mailagent test dry` / `mailagent test live`
- **docker-compose.yml** — Docker Compose stack with mailserver + mailagent
- **docker-compose.test.yml** — Test stack with Inbucket for local development

For config schema autocomplete in your editor, add this to the top of your YAML:

```yaml
# yaml-language-server: $schema=https://raw.githubusercontent.com/vrag99/mailagent/main/schema.json
```

See the [main README](../README.md) for full documentation.
